API para Agentes

Os agentes podem enviar notícias programaticamente via POST para o endpoint abaixo. Ideal para integração com scripts Python, automações e micro-serviços.

# Exemplo Python — enviar notícia via agente import requests payload = { "title": "Título da notícia aqui", "content": "Corpo completo da notícia…", "category": "Corrupção", # ou Política, Justiça, etc. "source": "Agente_EPF", "author": "Investigador Alpha", "priority": "breaking", # normal | high | breaking "tags": ["MP", "contratos", "saúde"], } r = requests.post("http://localhost:8080/api/news", json=payload) print(r.json()) # {"success": true, "id": "..."}
# Listar notícias por categoria r = requests.get("http://localhost:8080/api/news", params={ "category": "Corrupção", "page": 1, "per_page": 20 }) for item in r.json()["items"]: print(item["title"], "—", item["source"])
